Git

建立git仓库

cd到本地项目根目录下

执行命令：git init

添加和提交

  • 添加到暂存区

    添加文件：git add <filename>

    添加全部：git add * or git add .

  • 提交改动

    git commit -m "代码提交信息"

推送改动

  • 将仓库连接到远程服务器（在github上创建repository，获得url地址）

    git remote add origin <server>

  • 提交到远端仓库：

    git push -u origin master

    （可以把master换成其他分支）

分支

  • 创建分支并切换过去：

    git checkout -b <branch>

  • 切换分支：

    git checkout <branch>

  • 删除分支：

    git branch -d <branch>

  • 推送分支

    git push origin <branch>

更新与合并

  • 更新本地仓库至最新改动：

    git pull

  • 合并其他分支到当前分支：

    git merge <branch>

    （可能出现冲突，待完善）

其他

查看本地仓库历史记录：git log

显示工作目录和暂存区的状态 ：git status
